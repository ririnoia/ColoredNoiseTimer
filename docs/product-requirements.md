# Product Requirements: ColoredNoiseTimer

## 1. 概要

ColoredNoiseTimer は、色名ノイズを聞きながらポモドーロタイマーで集中できるWebアプリです。

## 2. 目的

ユーザーが余計な操作をせず、すぐにノイズ再生とタイマーを開始できる状態を作ること。

## 3. ターゲット

- 勉強に集中したい人
- 作業に集中したい人
- 開発・執筆・読書などで環境音を使いたい人
- シンプルな集中ツールを求める人

## 4. MVP範囲

### 必須

- ホームページ1枚
- ノイズ選択
- ノイズ再生 / 停止
- 音量調整
- ポモドーロタイマー
- 集中 / 休憩切り替え
- タイマー開始 / 停止
- タイマーリセット
- 集中時間・休憩時間の変更
- タイマー終了音ON/OFF
- ページタイトル更新
- スマホ対応

### MVPではやらない

- ログイン
- アカウント管理
- 問い合わせフォーム
- 決済
- 投げ銭
- バックエンドAPI
- DB保存
- 管理画面
- 広告の実表示

## 5. 将来追加候補

- ページ下部の小さな広告表示
- ノイズ種類追加
- プリセット追加
- 作業履歴
- PWA対応
- 多言語対応
- テーマ切り替え

## 6. 画面表示要素

- アプリ名: ColoredNoiseTimer
- 再生中のノイズ名
- ノイズ選択UI
- 音量スライダー
- タイマー
- 開始 / 停止ボタン
- 音声の再生 / 停止ボタン
- リセットボタン
- 集中 / 休憩の切り替え
- 集中時間入力
- 休憩時間入力
- タイマー終了音ON/OFF
- 将来広告用エリア

## 7. 初期設定

- Focus: 25分
- Break: 5分
- Timer end sound: ON
- Default noise: White Noise
- Volume: 50%

## 8. localStorageで保存してよい項目

- 最後に選択したノイズ
- 音量
- Focus時間
- Break時間
- タイマー終了音ON/OFF

## 9. 非機能要件

- スマホで使いやすいこと
- ページ読み込みが軽いこと
- UIが複雑にならないこと
- 主要操作が1画面内で完結すること
- 音声再生はユーザー操作後に開始すること
- ブラウザの自動再生制限を考慮すること
